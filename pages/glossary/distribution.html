📤 Glossary of Distribution 📤 Glossário de Distribuição

TermTermo DefinitionDefinição
CDNCDN Distributed network of servers that delivers files and digital content quickly, reducing latency and download times.Rede distribuída de servidores que entrega arquivos e conteúdos digitais de forma rápida, reduzindo latência e carregamento dos downloads.
CDN P2P HybridCDN Peer-to-Peer Hybrid System combining traditional CDN delivery with P2P technology to optimize distribution and reduce bandwidth costs.Sistema que combina entrega via CDN tradicional com tecnologia P2P para otimizar distribuição e reduzir custos de banda.
Direct Download Link (DDL)Link Direto Method of downloading where the file is accessed directly via HTTP or HTTPS link, usually hosted on web servers or file hosting services.Método de download onde o arquivo é acessado diretamente por um link HTTP ou HTTPS, geralmente hospedado em servidores web ou de hospedagem de arquivos.
DownloadDownload Process of receiving files from a server or another user to a local device.Processo de receber arquivos de um servidor ou outro usuário para um dispositivo local.
FTPFTP (File Transfer Protocol) Protocol for transferring files over the internet. Allows upload and download directly between computers connected to FTP servers.Protocolo para transferência de arquivos pela internet. Permite upload e download direto entre computadores conectados a servidores FTP.
File HostingHospedagem de Arquivos Online services that store files so users can download them via direct links; can be free or paid services.Serviços online que armazenam arquivos para que possam ser baixados por usuários via links diretos, podendo ser gratuitos ou pagos.
LeechLeech User who downloads files from the P2P network but shares little or nothing back, harming network efficiency.Usuário que baixa arquivos na rede P2P, mas que compartilha pouco ou nada, prejudicando a eficiência da rede.
MirrorMirror Exact copy of a file or website hosted on different servers to distribute traffic and increase content availability.Cópia exata de um arquivo ou site hospedada em servidores diferentes para distribuir o tráfego e aumentar a disponibilidade do conteúdo.
PeerPeer A BitTorrent client or user in a P2P network who is connected and sharing files.Um cliente BitTorrent ou usuário em uma rede P2P que está conectado e compartilhando arquivos.
Port ForwardingPort Forwarding Technique to open specific ports on a router to allow external connections, improving connectivity in P2P networks.Técnica para abrir portas específicas no roteador para permitir conexões externas, melhorando a conectividade em redes P2P.
SeedSeed User or server that has a complete copy of the file and shares it in the P2P network.Usuário ou servidor que possui a cópia completa do arquivo e o distribui na rede P2P.
SeedboxSeedbox Dedicated server used to store and distribute torrent files with high speed and availability.Servidor dedicado usado para armazenar e distribuir arquivos torrent com alta velocidade e disponibilidade.
SwarmSwarm Group of all peers and seeds sharing the same file in a P2P network.Conjunto de todos os peers e seeds compartilhando o mesmo arquivo em uma rede P2P.
TrackerTracker Server that coordinates communication between users in P2P networks, helping locate peers and organize file distribution.Servidor que coordena a comunicação entre usuários em redes P2P, ajudando a localizar peers e organizar a distribuição dos arquivos.
UploadUpload Process of sending files from a local device to a server or another user on the network.Processo de enviar arquivos de um dispositivo local para um servidor ou outro usuário na rede.
UsenetUsenet Global discussion and file sharing system using news servers, widely used for file distribution in newsgroups.Sistema global de discussão e compartilhamento de arquivos usando servidores de notícias, muito usado para distribuição de arquivos em newsgroups.
