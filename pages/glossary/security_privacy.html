🛡️ Glossary of Security & Privacy 🛡️ Glossário de Segurança & Privacidade

TermTermo DefinitionDefinição
2FA (Two-Factor Authentication)2FA (Autenticação de Dois Fatores) Authentication method requiring two different factors to gain access, enhancing security.Método de autenticação que exige dois fatores diferentes para acesso, aumentando a segurança.
AntivirusAntivírus Software that detects and removes malware and other threats from a system.Software que detecta e remove malwares e outras ameaças do sistema.
Multi-factor Authentication (MFA)Autenticação Multifator (MFA) Verification process using more than one method to confirm user identity.Processo de verificação que usa mais de um método para confirmar a identidade do usuário.
EncryptionCriptografia Process of encoding data to prevent unauthorized access.Processo de codificar dados para impedir acesso não autorizado.
Data LeakVazamento de Dados Unauthorized disclosure of sensitive or private information.Divulgação não autorizada de informações sensíveis ou privadas.
FirewallFirewall System that monitors and controls network traffic based on security rules.Sistema que monitora e controla o tráfego de rede baseado em regras de segurança.
HTTPSHTTPS Protocol for secure communication on the web using SSL/TLS encryption.Protocolo para comunicação segura na web, usando criptografia SSL/TLS.
KeyloggerKeylogger Software or hardware that records keystrokes for spying or data theft.Software ou hardware que grava as teclas digitadas para espionagem ou roubo de dados.
Penetration TestingTeste de Penetração Controlled simulation of attacks to identify system vulnerabilities.Simulação controlada de ataque para identificar vulnerabilidades em sistemas.
PhishingPhishing Fraudulent technique to obtain personal information by pretending to be a trusted entity.Técnica fraudulenta para obter informações pessoais, fingindo ser uma entidade confiável.
PrivacyPrivacidade Rights and practices to protect personal data and control its use.Direito e práticas para proteger dados pessoais e controlar seu uso.
Privacy PolicyPolítica de Privacidade Document explaining how an organization collects, uses, and protects user data.Documento que explica como uma organização coleta, usa e protege dados dos usuários.
ProxyProxy Server acting as an intermediary between user and the internet to protect identity and access.Servidor que atua como intermediário entre o usuário e a internet para proteger identidade e acesso.
RansomwareRansomware Malware type that blocks data access until a ransom is paid.Tipo de malware que bloqueia o acesso aos dados até que um resgate seja pago.
Information SecuritySegurança da Informação Practice of protecting data against unauthorized access, use, disclosure, or destruction.Prática de proteger dados contra acesso, uso, divulgação ou destruição não autorizados.
Strong PasswordSenha Forte Password difficult to guess, usually combining letters, numbers, and symbols.Senha difícil de ser descoberta, normalmente combinando letras, números e símbolos.
TokenToken Device or code used for secure authentication.Dispositivo ou código usado para autenticação segura.
VPN (Virtual Private Network)VPN (Rede Privada Virtual) Private network creating a secure and encrypted connection over the internet.Rede privada que cria uma conexão segura e criptografada pela internet.
Zero TrustZero Trust Security model that assumes no entity, internal or external, is trusted by default.Modelo de segurança que assume que nenhuma entidade, interna ou externa, é confiável por padrão.
End-to-End EncryptionCriptografia de Ponta a Ponta Encryption method where only sender and receiver can read the data.Método de criptografia onde somente o remetente e o destinatário podem ler os dados.
MalwareMalware Malicious software designed to cause damage or unauthorized access to systems.Software malicioso projetado para causar danos ou acesso não autorizado a sistemas.
SpywareSpyware Software that collects user information without their knowledge.Software que coleta informações do usuário sem seu conhecimento.
