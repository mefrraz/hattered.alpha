🛠️ Glossary of Content Mod 🛠️ Glossário de Modificação

TermTermo DefinitionDefinição
Adding SubtitlesAdicionar Legendas Process of inserting subtitles into videos to make them accessible in different languages.Processo de inserir legendas em vídeos para torná-los acessíveis em diferentes idiomas.
CompressorCompressor Software or tool that reduces file size to ease distribution and storage.Software ou ferramenta que reduz o tamanho dos arquivos para facilitar distribuição e armazenamento.
EncodingCodificação Process of converting audio or video files into a specific format, often for compression.Processo de converter arquivos de áudio ou vídeo para um formato específico, frequentemente para compressão.
CrackCrack Modification that removes software protections to allow unauthorized use.Modificação que remove proteções de software para permitir uso não autorizado.
PatchCorreção (Patch) Small program or file that fixes bugs, adds features, or modifies software.Pequeno programa ou arquivo que corrige erros, adiciona funcionalidades ou modifica um software.
RippingExtração Process of copying content from physical media like CDs or DVDs into digital files.Processo de copiar conteúdo de mídias físicas, como CDs ou DVDs, para arquivos digitais.
File MergingFusão de Arquivos Combining multiple separate files into a single file for easier use.Combinar vários arquivos separados em um único arquivo para facilitar o uso.
IndexingIndexação Creating a file or list that organizes and facilitates access to content.Criação de um arquivo ou lista que organiza e facilita o acesso ao conteúdo.
OCR (Optical Character Recognition)OCR (Reconhecimento Óptico de Caracteres) Technology that converts images of text into editable and searchable text.Tecnologia que converte imagens de texto em texto editável e pesquisável.
Re-encodeRe-encode Process of encoding a file again, usually to change format or quality.Processo de codificar novamente um arquivo, geralmente para alterar o formato ou qualidade.
RemuxRemux Process of changing the container of a media file without altering the original content.Processo de trocar o container de um arquivo de mídia sem alterar o conteúdo original.
SplittingSegmentação Dividing a large file into smaller parts for easier download or transfer.Dividir um arquivo grande em partes menores para facilitar o download ou transferência.
Subtitle SyncingSincronização de Legendas Timing adjustment of subtitles to match the video audio.Ajuste do tempo das legendas para que coincidam com o áudio do vídeo.
TranscodingTranscodificação Conversion of media files from one format to another, possibly altering quality.Conversão de arquivos de mídia de um formato para outro, possivelmente alterando a qualidade.
TranslationTradução Process of converting text or audio from one language to another.Processo de converter texto ou áudio de um idioma para outro.
UnpackingUnpacking Removing protection or compression from files to allow access to original content.Remover a proteção ou compactação de arquivos para permitir acesso ao conteúdo original.
