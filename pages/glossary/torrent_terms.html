🧲 Glossary of Torrent 🧲 Glossário de Torrent

TermTermo DefinitionDefinição
AvailabilityDisponibilidade Number of full copies of a file (or set of files) available to the client. Each seed adds 1.0.Número de cópias completas de um arquivo (ou conjunto de arquivos) disponíveis para o cliente. Cada seed adiciona 1.0.
BitTorrentBitTorrent The most widely used peer-to-peer filesharing system. Decentralized and splits downloads into pieces from peers.O método mais utilizado de compartilhamento peer-to-peer. Sistema descentralizado que organiza downloads em pedaços distribuídos entre peers.
ConnectabilityCapacidade de Conexão The state of being connectable with an open port allowing peers to connect and share data.Estado de poder receber conexões, ou seja, ter uma porta aberta para outros peers se conectarem e compartilharem dados.
Cross SeedCross Seed When the same torrent is on different trackers, you can download from one and seed to both.Quando o mesmo torrent está em trackers diferentes, pode-se baixar de um e seedar em ambos.
DHTDHT Distributed Hash Table: a peer-to-peer network to find peers without a central tracker.Distributed Hash Table: uma rede peer-to-peer que ajuda a encontrar peers sem precisar de um tracker centralizado.
FreeleechFreeleech Torrents that don’t count toward your downloaded total, only upload counts toward your ratio.Torrents que não contam para o total baixado na sua conta, apenas o upload conta para a sua proporção (ratio).
HashHash Digital fingerprint used to verify data integrity. If verification fails, data is discarded and re-requested.Impressão digital do torrent usada para verificar a integridade dos dados. Se a verificação falha, o dado é descartado e pedido novamente.
Hit-and-run (HnR)Hit-and-run (HnR) Downloading a file and seeding as little as possible, harming the swarm.Baixar um arquivo e seedar o mínimo possível, prejudicando a comunidade.
IP AddressEndereço IP Unique numeric identifier for devices on an IP network. Can be dynamic or static.Identificador único numérico para dispositivos numa rede que usam protocolo IP. Pode ser dinâmico ou estático.
Leech / LeechingLeech / Leeching Peer without 100% of the file or one who downloads more than uploads, negatively affecting the ratio.Peer que não tem 100% do arquivo ou que baixa mais do que faz upload, prejudicando a proporção na rede.
MagnetMagnet Link that enables downloading files directly from P2P networks without the .torrent file.Link que permite baixar arquivos diretamente de redes P2P sem precisar do arquivo .torrent.
PeerPeer Instance of a BitTorrent client connected to others for data sharing. Can be downloader or seed.Instância de cliente BitTorrent conectado a outros para compartilhar dados. Pode ser downloader ou seed.
Port ForwardingPort Forwarding Opening router ports to improve connectivity and speed in torrent sharing.Abrir portas no roteador para melhorar a conectividade e velocidade do compartilhamento de torrents.
Private TrackerPrivate Tracker Private tracker that restricts access via invitation or interview, promoting high quality and community control.Tracker privado que limita o acesso via convite ou entrevista, promovendo alta qualidade e controle da comunidade.
Public TrackerPublic Tracker Public tracker open for anyone to add to torrents.Tracker público, acessível a qualquer pessoa para adicionar a torrents.
RatioRatio Ratio between uploaded and downloaded data by a user. Above 1.0 is positive, below is negative.Proporção entre dados enviados e recebidos pelo usuário. Acima de 1.0 indica contribuição positiva; abaixo, negativa.
Seed / SeedingSeed / Seeding A peer that has 100% of the file and uploads to others.Um peer que tem 100% do arquivo e está enviando dados para outros peers.
SeedboxSeedbox Rented server used to download and seed torrents, usually with high speed and uptime.Servidor alugado usado para baixar e seedar torrents, geralmente com alta velocidade e disponibilidade.
SnatchSnatch Term for when a torrent has been fully downloaded by the user.Termo para quando um torrent foi completamente baixado pelo usuário.
Super-SeedingSuper-Seeding Algorithm to help new downloaders become seeders faster but can fail if only one downloader exists.Algoritmo que ajuda novos downloaders a virar seeders rapidamente, mas pode falhar se houver apenas um downloader.
SwarmSwarm Group of all peers and seeds sharing the same content.Conjunto de todos os peers e seeds compartilhando o mesmo conteúdo.
TorrentTorrent File containing metadata about files to distribute, including tracker list and peer locations.Arquivo que contém metadados sobre arquivos a distribuir, incluindo lista de trackers e localização dos peers.
TrackerTracker Server assisting communication between peers, helping locate and coordinate file transfer.Servidor que auxilia a comunicação entre peers, ajudando a localizar e coordenar a transferência dos arquivos.
Tracker ScrapeTracker Scrape Query to tracker to get info on availability and number of peers for a torrent.Consulta feita ao tracker para obter informações sobre a disponibilidade e número de peers para um torrent.
Upload SlotUpload Slot Number of simultaneous connections a seed can accept to upload data.Número de conexões simultâneas que um seed pode aceitar para enviar dados.
