🎮 Glossary of Software & Games 🎮 Glossário de Softwares & Jogos

TermTermo DefinitionDefinição
CFWCFW Custom firmware altering original system software to run unofficial software.Firmware personalizado que modifica o software original do console para rodar software não oficial.
Clean Steam FilesClean Steam Files Game files not cracked, downloaded directly from Steam, used to apply cracks or emulators.Arquivos de jogo não crackeados, exatamente como baixados do Steam, usados para aplicar cracks ou emuladores.
CrackCrack File that bypasses DRM protection to use software/game without paying. May be flagged by antivirus.Arquivo que permite burlar a proteção DRM para usar o software/jogo sem pagar. Pode ser detectado como suspeito.
DenuvoDenuvo Anti-tamper DRM tech designed to protect games from cracking and illegal copying.Tecnologia anti-tamper e DRM que protege jogos contra crack e cópia ilegal.
DLCDLC Downloadable content for games distributed online by the publisher.Conteúdo adicional para jogos, distribuído online pelo desenvolvedor.
DLC UnlockerDLC Unlocker Software tool that unlocks DLC content without purchase.Ferramenta que desbloqueia DLCs sem necessidade de compra.
EmulatorEmulator Software that allows running games from other platforms on PC or emulators.Programa que permite rodar jogos de outras plataformas no PC ou emuladores.
File-typesFile-types Detailed index of file formats used by different consoles.Índice detalhado dos formatos de arquivos usados em diferentes consoles.
HomebrewHomebrew Modding or hacking consoles to run unofficial software.Modificação do console para rodar software não oficial, também chamado de modding ou jailbreaking.
ISOISO Optical disc image file containing all data from original disc, used for games and software.Imagem de disco óptico que contém todos os dados do disco original, usada para jogos e software.
KeygenKeygen Program that generates activation keys to unlock software.Programa que gera chaves de ativação para desbloquear software.
LoaderLoader Program that modifies a game's executable to bypass copy protection.Programa que modifica o executável do jogo para burlar proteção contra cópia.
ModMod Community-created modification altering or adding content to original game.Modificação criada pela comunidade para alterar ou adicionar conteúdo ao jogo original.
PatchPatch Small program that modifies the machine code of another program.Pequeno programa que modifica o código de outro programa.
Pre-crackedPre-cracked Game or software files already cracked, simplifying installation.Software ou jogo já crackeado, facilitando a instalação.
Pre-install / PortablePre-install / Portable Games or software pre-installed in folders, usable without installation.Software ou jogo já instalado em pasta, podendo ser usado sem instalação.
Release GroupRelease Group Group responsible for cracking, packaging, and distributing pirated games.Grupo responsável por crackear, empacotar e distribuir jogos piratas.
RepackRepack Compressed games for easier download, often pre-cracked.Jogos compactados para facilitar o download, geralmente pré-crackeados.
ROMROM File containing game data used with emulators.Arquivo contendo dados de jogo para uso em emuladores.
Steam Emulator (Emu)Steam Emulator (Emu) Software allowing users to play Steam games without the official Steam client.Software que permite jogar jogos Steam sem o cliente oficial Steam.
TrainerTrainer Cheat app that alters game features (god mode, unlimited ammo, etc.).Aplicativo de trapaça que altera características do jogo (vida infinita, munição, etc.).
